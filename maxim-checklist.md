# Чеклист для Maxim — заказ 2 NSK IP + 1 SPB VPS

## Шаг 1: 2 публичных IP в NSK-1 (для текущего сервера)

**URL:** https://timeweb.cloud/my/floating-ip

1. Нажми **«Создать»**
2. **Регион:** Новосибирск (NSK-1) — та же зона где текущий VPS
3. **Создать**
4. Повтори ещё раз — должно быть **2 IP в списке** (пока не привязанные)
5. **Не привязывай через UI** — пусть саппорт сделает batch-привязку
6. **Скопируй оба IP** и сообщи мне сюда + в тикет Timeweb

**Стоимость:** 180 ₽/мес × 2 = 360 ₽/мес. За 2-3 дня теста ~24-36 ₽.

## Шаг 2: Минимальный VPS в SPB-3

**URL:** https://timeweb.cloud/my/servers/create

1. Нажми **«Создать»**
2. **Регион:** Санкт-Петербург (SPB-3)
3. **Конфигурация:** **самый дешёвый тариф** (обычно ~200-300 ₽/мес — посмотри какой минимальный CPU/RAM)
4. **ОС:** **Ubuntu 24.04** (важно — такая же как production VPS)
5. **SSH-ключ:** прикрепи свой существующий ключ (тот же что у `vps` alias) — чтобы я мог SSH-проверить если нужно
6. **Имя сервера:** `vps-monitor-spb-test` (или любое осмысленное)
7. **Создать**
8. Дождись пока сервер появится со статусом «Запущен» — у него будет публичный IP уже привязан автоматически
9. **Скопируй IP** этого SPB VPS и сообщи мне

**Стоимость:** ~200-300 ₽/мес = ~13-20 ₽ за 2 дня теста.

**Что НЕ нужно делать на SPB VPS:**
- ❌ Не ставить nginx
- ❌ Не настраивать сайты
- ❌ Не копировать данные
- ❌ Не привязывать домены

SSH-демон работает с порта 22 по умолчанию — этого достаточно для нашего TCP-probe.

## Шаг 3: После заказа — сообщить мне

Просто пришли мне в чате 3 IP-адреса:
```
NSK candidate 1: XXX.XXX.XXX.XXX
NSK candidate 2: YYY.YYY.YYY.YYY
SPB candidate 3: ZZZ.ZZZ.ZZZ.ZZZ
```

И отдельно сообщи в тикет Timeweb 2 NSK IP с просьбой batch-привязки + выбранным временем reboot.

## Шаг 4: Что произойдёт дальше (на моей стороне)

1. Я обновлю YC function через UI editor — добавлю `VPS_CANDIDATE_IPS=IP1,IP2,IP3`
2. Function начнёт каждые 5 минут пробить все 3 IP через TCP 22 (+ TCP 443)
3. Данные пойдут в `logs/probes-candidates.csv`
4. Через 24-48 часов проанализируем — какой IP/зона стабильнее
5. Принимаем решение по миграции

| 2 NSK IP | 12 ₽ | 24 ₽ |
